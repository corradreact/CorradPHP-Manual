# System Settings - Overview

## Purpose

System Settings adalah pusat konfigurasi sistem PHP84 CORRAD. Di sini administrator dapat mengatur berbagai parameter sistem, koneksi database, data referensi, LDAP, dan JWT.

## When to Use

Gunakan modul ini untuk:
- Mengkonfigurasi parameter sistem utama
- Mengelola koneksi database
- Mengatur data referensi
- Konfigurasi LDAP/AD untuk authentication
- Setup JWT untuk API security

## Key Features

1. **System Configuration** - Pengaturan dasar sistem
2. **Database Connections** - Manajemen koneksi database
3. **Reference Data** - Data master/lookup
4. **LDAP Editor** - Konfigurasi LDAP/Active Directory
5. **JWT Server** - Token configuration untuk API

## Important Notes

⚠️ **Perhatian:**
- Perubahan di sini dapat mempengaruhi seluruh sistem
- Hanya administrator terlatih yang seharusnya mengubah pengaturan ini
- Selalu backup konfigurasi sebelum membuat perubahan besar

ℹ️ **Catatan:**
- Beberapa setting memerlukan restart sistem untuk berlaku
- Dokumentasi detail tersedia di submenu masing-masing

## Expected Result

Sistem dikonfigurasi sesuai kebutuhan organisasi dan dapat diintegrasikan dengan sistem external.
