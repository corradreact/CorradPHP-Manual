# Cara Login ke Sistem

## Purpose

Halaman ini menjelaskan bagaimana cara masuk (login) ke sistem PHP84 CORRAD menggunakan username dan password.

## When to Use

Gunakan panduan ini setiap kali anda pertama kali menggunakan sistem atau setelah logout.

## Steps

1. Buka browser web anda (Chrome, Firefox, Safari, atau Edge)
2. Masukkan URL sistem: **https://php84.corrad.my/**
3. Tunggu halaman login dimuat sepenuhnya
4. Masukkan **Username** anda di field pertama
5. Masukkan **Password** anda di field kedua
6. Periksa apakah kapslock aktif atau tidak (jika perlu)
7. Klik tombol **Login** untuk masuk

## Important Notes

⚠️ **Perhatian Penting:**
- Jangan pernah bagikan username dan password anda kepada siapa pun
- Pastikan anda logout saat selesai menggunakan sistem, terutama di komputer bersama
- Password bersifat case-sensitive (membedakan huruf besar dan kecil)

ℹ️ **Catatan:**
- Jika anda lupa password, klik link **Forgot Password** di halaman login
- Hubungi administrator jika anda tidak memiliki akun

## Expected Result

Setelah login berhasil, anda akan diarahkan ke halaman **Dashboard** yang menampilkan ringkasan sistem dan menu utama.

## Common Issues

### Login Gagal dengan Pesan "Invalid Username or Password"
- **Penyebab**: Username atau password salah
- **Solusi**: 
  - Periksa kembali username dan password
  - Pastikan tidak ada spasi di awal atau akhir
  - Pastikan caps lock tidak aktif (kecuali password memang uppercase)

### Halaman Login Tidak Bisa Dibuka
- **Penyebab**: Masalah koneksi internet atau server tidak aktif
- **Solusi**:
  - Periksa koneksi internet anda
  - Coba buka halaman lain untuk verifikasi koneksi
  - Hubungi administrator jika server sedang maintenance

### Login Berhasil tapi Halaman Blank/Error
- **Penyebab**: Masalah dengan browser cache atau session
- **Solusi**:
  - Bersihkan cache browser (Ctrl+Shift+Delete)
  - Tutup semua tab dan buka ulang
  - Coba gunakan browser berbeda

### Akun Terkunci Setelah Beberapa Kali Login Gagal
- **Penyebab**: Sistem keamanan untuk mencegah brute force attack
- **Solusi**:
  - Tunggu beberapa menit sebelum mencoba lagi
  - Hubungi administrator untuk membuka kunci akun
