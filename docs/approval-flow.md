# Approval Flow

Penjelasan aliran kelulusan dalam sistem.

<div class="main-card">

## Aliran kelulusan biasa

1. Pengguna menghantar permohonan atau rekod.
2. Rekod dihantar kepada penyemak atau penyelia.
3. Penyelia menyemak maklumat.
4. Penyelia boleh **Approve**, **Reject** atau **Request Changes**.
