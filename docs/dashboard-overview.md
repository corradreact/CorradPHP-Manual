# Dashboard Overview

Dashboard adalah halaman utama selepas log masuk. Ia memaparkan ringkasan data dan jalan pintas ke modul penting.

<div class="main-card">

## Komponen utama dashboard

- **Header ringkas** dengan nama pengguna dan notifikasi.
- **Kad ringkasan** untuk jumlah rekod, kelulusan, dan statistik.
- **Pautan cepat** ke modul utama seperti `Training`, `Budget`, atau `Requests`.
- **Grafik ringkas** untuk pemantauan prestasi.

## Cara guna dashboard

- Semak **Ringkasan Hari Ini** untuk kemas kini cepat.
- Klik **pautan cepat** untuk akses pantas ke fungsi utama.
- Gunakan menu atas atau sisi untuk navigasi penuh.
