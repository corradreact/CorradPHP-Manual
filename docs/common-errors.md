# Common Errors

Senarai masalah biasa yang dihadapi pengguna dan cara mengatasinya.

<div class="main-card">

## Masalah biasa

- `Invalid username or password` — semak semula kredensial.
- `Session expired` — log keluar dan log masuk semula.
- `File upload failed` — semak saiz atau jenis fail.
- `Permission denied` — hubungi pentadbir jika anda tidak mempunyai akses.
