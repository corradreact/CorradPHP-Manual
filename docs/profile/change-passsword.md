# Tukar Kata Laluan

::: info Tujuan
Membantu pengguna menukar kata laluan akaun mereka dengan selamat. Panduan ini menerangkan cara membuka bahagian tukar kata laluan, mengisi maklumat kata laluan semasa dan baru, dan menyimpan perubahan.
:::

## 1. Buka Halaman Tukar Kata Laluan

::: tip Langkah 1
1) Log masuk ke dalam sistem.
2) Pergi ke halaman **Profile**.
3) Pilih pilihan **Change Password** atau **Tukar Kata Laluan**.
:::

## 2. Isi Maklumat Kata Laluan

Pada halaman tukar kata laluan, anda akan melihat tiga medan:
- **Current Password**: masukkan kata laluan semasa anda.
- **New Password**: masukkan kata laluan baharu.
- **Confirm New Password**: taip semula kata laluan baharu untuk pengesahan.

::: tip Langkah 2
1) Ketik kata laluan semasa dalam medan **Current Password**.
2) Isikan kata laluan baharu dalam medan **New Password**.
3) Sahkan kata laluan baru dengan memasukkannya semula dalam medan **Confirm New Password**.
:::

## 3. Tips Kata Laluan Selamat

Pastikan kata laluan baharu memenuhi syarat berikut:
- Minimum 12 aksara.
- Mempunyai gabungan huruf besar dan kecil.
- Mengandungi sekurang-kurangnya satu nombor.
- Mengandungi aksara khas seperti `@`, `#`, atau `!`.

## 4. Simpan Perubahan

::: tip Langkah 3
1) Klik butang **Save** untuk mengemas kini kata laluan.
2) Jika anda tidak mahu meneruskan, klik **Cancel**.
:::

### Selepas Menyimpan
- Sistem akan mengemaskini kata laluan anda.
- Anda mungkin perlu log masuk semula dengan kata laluan baharu.

::: warning Nota
- Jangan berkongsi kata laluan anda dengan orang lain.
- Pastikan kata laluan baharu berbeza daripada kata laluan lama.
- Jika anda terlupa kata laluan semasa, gunakan pilihan **Forgot Password** jika disediakan atau hubungi pentadbir.
:::
