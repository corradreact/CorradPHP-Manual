# Export / Download Data

Langkah mengeksport atau memuat turun data dari sistem.

<div class="main-card">

## Proses eksport data

1. Buka modul yang mempunyai data yang ingin dieksport.
2. Cari butang **Export** atau **Download**.
3. Pilih format yang disokong, contohnya `XLSX`, `CSV`, atau `PDF`.
4. Klik **Download**.
