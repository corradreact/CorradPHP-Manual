# Edit Existing Record

Langkah untuk mengemas kini rekod sedia ada.

<div class="main-card">

## Panduan penyuntingan

1. Cari rekod yang ingin diubah menggunakan carian atau penapis.
2. Klik butang **Edit** atau ikon pensel.
3. Buat perubahan pada medan yang diperlukan.
4. Klik **Update** atau **Save**.
