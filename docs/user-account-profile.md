# User Account / Profile

Bahagian profil pengguna memberi akses kepada maklumat akaun, tetapan, dan pilihan peribadi.

<div class="main-card">

## Apa yang anda boleh lakukan

- Kemaskini maklumat akaun.
- Tukar kata laluan.
- Semak peranan dan kebenaran pengguna.
- Ubah tetapan notifikasi.

## Langkah ringkas

1. Klik ikon profil di sudut atas.
2. Pilih **Profile** atau **Account Settings**.
3. Kemas kini maklumat yang diperlukan.
4. Klik **Save**.
