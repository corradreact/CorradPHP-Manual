# FAQ

Soalan lazim berkaitan penggunaan sistem PHP84 CORRAD.

<div class="main-card">

## Soalan dan jawapan

**Q: Bagaimana saya dapat reset kata laluan?**

A: Hubungi pentadbir sistem atau gunakan pautan reset jika tersedia.

**Q: Bagaimana cara cari rekod lama?**

A: Gunakan fungsi carian dan tapis dengan tarikh atau status.

**Q: Siapa yang boleh approve permohonan?**

A: Hanya pengguna dengan peranan penyelia atau pentadbir.

**Q: Bolehkah saya eksport data ke Excel?**

A: Ya, pilih `Export` dan pilih format `XLSX` atau `CSV`.
