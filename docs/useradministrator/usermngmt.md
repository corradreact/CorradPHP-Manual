# Pengurusan Pengguna (User Management)

::: info Tujuan
Membantu admin mengurus pengguna dalam sistem. Panduan ini menerangkan cara melihat senarai pengguna, menambah pengguna baharu, mengedit maklumat pengguna, dan memadam pengguna.
:::

## 1. Senarai Pengguna
### Akses Menu User Management

::: tip Langkah 1
1) Log masuk ke dalam sistem sebagai admin.
2) Pilih menu **User Management** atau **Pengurusan Pengguna**.
3) Sistem akan memaparkan senarai pengguna dengan maklumat seperti **USERNAME**, **FULL NAME**, **EMAIL**, **USER TYPE**, **STATUS** dan **ACTIONS**.
:::
![Senarai Pengguna]
## 2. Cari dan Tapis Pengguna
Gunakan kotak carian di bahagian atas untuk mencari pengguna mengikut nama pengguna, nama penuh atau email.

## 3. Menambah Pengguna Baharu

::: tip Langkah 2
1) Klik butang **New User** atau ikon tambah pengguna.
2) Isikan maklumat pengguna baharu seperti username, nama penuh, email dan jenis pengguna.
3) Klik butang **Tambah** atau **Simpan**.
:::
![Tambah Pengguna]

4) Sistem akan memaparkan mesej berjaya seperti:
**"Pengguna berjaya ditambah ke pangkalan data"**
![Success]

5) Pengguna baru akan terus muncul dalam senarai.

## 4. Mengemaskini Maklumat Pengguna
Untuk mengubah maklumat pengguna:

::: tip Langkah 3
1) Klik butang **Edit** di sebelah pengguna yang hendak dikemaskini.
2) Ruangan input akan aktif supaya anda boleh mengemaskini maklumat.
3) Masukkan maklumat baharu dan klik **Simpan**.
:::
![Edit Pengguna]
![Simpan]
4) Sistem akan memaparkan mesej:
**"Pengguna berjaya dikemaskini"**
![Success3]
## 5. Memadam Pengguna
Jika pengguna tidak lagi perlu berada dalam sistem:

::: tip Langkah 4
1) Klik butang **Padam** di sebelah pengguna.
2) Sahkan tindakan jika diminta.
:::
![Padam]

3) Sistem akan memaparkan mesej:
**"Pengguna dipadam dari pangkalan data"**
![Success4]

::: warning Nota
- Pastikan maklumat pengguna adalah tepat sebelum disimpan kerana ia akan digunakan dalam rekod sistem.
- Tindakan memadam pengguna adalah kekal dan tidak boleh dibatalkan atau dipinda semula.
- Semak semula status pengguna dan jenis pengguna sebelum membuat perubahan.
:::
