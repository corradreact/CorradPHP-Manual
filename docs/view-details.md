# View Details

Panduan melihat maklumat penuh rekod.

<div class="main-card">

## Langkah lihat maklumat rekod

1. Cari rekod yang ingin dilihat.
2. Klik tajuk atau butang **View**.
3. Semak maklumat penuh di halaman terperinci.
4. Jika perlu, gunakan `Back` untuk kembali ke senarai.
