# Search and Filter Data

Panduan mencari dan menapis data dalam sistem.

<div class="main-card">

## Cara guna carian dan penapis

1. Masukkan kata kunci dalam kotak carian.
2. Gunakan penapis tarikh, status, atau kategori.
3. Klik **Apply** untuk lihat hasil terkini.
4. Reset penapis jika mahu mula semula.

## Petua carian

- Gunakan istilah yang spesifik seperti `Nombor Permohonan`.
- Semak semula tarikh atau status jika tiada hasil ditemui.
- Gunakan kombinasi penapis untuk mengecilkan hasil.
