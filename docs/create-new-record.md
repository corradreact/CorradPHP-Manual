# Create New Record

Panduan membuat rekod baru dalam sistem.

<div class="main-card">

## Langkah membuat rekod baru

1. Pergi ke modul yang sesuai di menu.
2. Klik butang **Create New** atau **Add Record**.
3. Isi borang dengan maklumat yang diperlukan.
4. Semak semula data.
5. Klik **Save** atau **Submit**.
