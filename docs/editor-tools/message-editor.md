# Message Editor

::: info Tujuan
Membantu admin mengurus teks mesej dalam sistem. Panduan ini menerangkan cara melihat senarai mesej, menambah mesej baru, mengedit mesej sedia ada dan memadam mesej.
:::

## 1. Buka Message Editor

::: tip Langkah 1
1) Log masuk ke dalam sistem sebagai admin.
2) Pilih menu **Editors & Tools** kemudian pilih **Message Editor**.
3) Sistem akan memaparkan papan pemuka Message Editor dengan statistik jenis mesej.
:::

## 2. Memahami Paparan Utama

Pada halaman utama anda akan melihat:
- **All Messages**: jumlah semua mesej.
- **Info**: jumlah mesej jenis maklumat.
- **Error**: jumlah mesej jenis ralat.
- Senarai mesej dengan kolum **Type**, **Name**, **Text** dan **Actions**.
- Kawalan carian untuk mencari mesej mengikut nama atau teks.

## 3. Cari Mesej

::: tip Langkah 2
1) Taip nama atau teks yang ingin dicari di medan **Search**.
2) Pilih jenis mesej dalam menu **Type** jika anda mahu tapis kepada Info atau Error sahaja.
3) Klik **Search** untuk melihat keputusan yang sepadan.
:::

## 4. Tambah Mesej Baharu

::: tip Langkah 3
1) Klik butang **+ New Message**.
2) Dalam tetingkap **New Message**, pilih jenis mesej sama ada **Info** atau **Error**.
3) Isi medan **Name** dengan kod mesej seperti `MSG_SAVE_SUCCESS`.
4) Isi medan **Text** dalam tab **BM** untuk teks Bahasa Malaysia dan tab **BI** untuk teks Bahasa Inggeris.
5) Klik **Save** untuk menyimpan mesej baharu.
:::

## 5. Edit Mesej

::: tip Langkah 4
1) Cari mesej yang ingin dikemaskini dalam senarai.
2) Klik ikon **Edit** di lajur **Actions**.
3) Dalam tetingkap **Edit Message**, ubah jenis, nama atau teks mesej seperti perlu.
4) Klik **Save** untuk mengemas kini mesej.
:::

## 6. Padam Mesej

::: tip Langkah 5
1) Cari mesej yang hendak dipadam.
2) Klik ikon **Delete** di lajur **Actions**.
3) Sahkan tindakan dengan memilih **Delete** dalam tetingkap pengesahan.
:::

### Nota Penghapusan
- Selepas dipadam, mesej tidak boleh dikembalikan.
- Pastikan mesej tidak lagi digunakan dalam sistem sebelum memadamnya.

::: warning Nota
- Gunakan nama mesej yang jelas dan konsisten supaya mudah dicari.
- Semak teks BM dan BI sebelum menyimpan untuk elak kesilapan bahasa.
- Elakkan memadam mesej penting tanpa semakan berkaitan impak aplikasi.
:::
