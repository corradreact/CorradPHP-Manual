# Translation Editor

::: info Tujuan
Membantu pentadbir mengurus terjemahan CORRAD UI merentas pelbagai bahasa. Panduan ini mengikuti gaya `Guide & Usage` dengan penjelasan ringkas, penggunaan jadual dan penyuntingan, pengurusan bahasa, serta petua terjemahan.
:::

## Overview

Translation Editor membenarkan pentadbir mentranslate elemen UI (menu, halaman, komponen, item, kawalan, mesej) terus dari paparan jadual. Setiap baris menunjukkan teks asal dan terjemahan untuk bahasa aktif.

Paparan ini juga memaparkan statistik penting seperti:
- jumlah bahasa
- jumlah terjemahan
- bahasa aktif
- top type terjemahan

### Quick Workflow
- Pilih jenis teks melalui tab jenis seperti **Index**, **Menu**, **Page**, **Component**, **Item**, **Control**, **Message**.
- Gunakan carian untuk menapis berdasarkan teks sumber.
- Klik sel dalam lajur bahasa untuk mengedit terjemahan inline.
- Klik ikon lampu untuk mendapat cadangan terjemahan automatik.
- Klik **Manage Languages** untuk tambah atau urus bahasa.

## Table & Editing

### Cari dan Tapis

- Gunakan kotak carian untuk mencari entri berdasarkan teks asal.
- Pilih tab jenis terjemahan untuk melihat hanya elemen tertentu.

### Penyuntingan Inline

- Klik pada sel terjemahan dalam lajur bahasa yang aktif.
- Edit terus teks Bahasa Melayu atau Bahasa Inggeris.
- Simpan perubahan jika sistem memerlukan.

### Auto Translation

- Klik ikon lampu (lightbulb) pada sel terjemahan untuk mendapatkan cadangan terjemahan automatik.
- Kaedah ini sesuai untuk mempercepat kerja, tetapi sentiasa semak hasil terjemahan sebelum menyimpan.

## Languages

### Urus Bahasa

- Klik **Manage Languages** untuk melihat senarai bahasa.
- Setiap bahasa menunjukkan bendera, nama, status dan bilangan terjemahan.

### Tambah Bahasa

- Klik **Add Language**.
- Isi **Language Name**, pilih **Flag**, dan masukkan **Language Code** seperti `ms`.
- Tandakan **Active** jika mahu bahasa tersebut didapati di editor.
- Klik **Save**.

### Edit Bahasa

- Klik ikon edit pada baris bahasa.
- Ubah nama, bendera atau kod bahasa.
- Simpan perubahan.

## Types & Tips

### Jenis Terjemahan

Gunakan tab jenis untuk memfokuskan pada elemen tertentu:
- **Index**
- **Menu**
- **Page**
- **Component**
- **Item**
- **Control**
- **Message**

### Petua Terjemahan

- Mulakan dengan memilih tab yang sesuai untuk kerja anda.
- Guna carian teks sumber apabila mencari frasa tertentu.
- Gunakan ikon lampu untuk cadangan automatik, kemudian semak semula terjemahan.
- Pastikan terjemahan mengekalkan makna asal dan gaya bahasa yang jelas.
- Semak terjemahan dalam konteks UI jika boleh untuk mengelakkan teks berlebihan.

::: warning Nota
- Terjemahan automatik boleh membantu, tetapi tidak menggantikan semakan manusia.
- Jangan simpan terjemahan yang masih belum disemak atau tidak lengkap.
- Tetapkan bahasa hanya yang diperlukan untuk mengurangkan kekeliruan.
:::
