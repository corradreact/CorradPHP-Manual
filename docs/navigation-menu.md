# Navigation Menu

Menu navigasi membantu anda bergerak antara modul dan fungsi utama sistem.

<div class="main-card">

## Struktur menu

- **Dashboard Admin**
- **Laporan Latihan**
- **Pengurusan Permohonan**
- **Tetapan & Pengurusan Emel**

Each section may contain submenu items, seperti:

- `Daftar Peserta Latihan`
- `Senarai Permohonan`
- `Urus Tempoh Permohonan`

## Petua navigasi

- Klik menu utama untuk berkembang submenu.
- Menu sisi menyokong akses cepat ke modul yang sering digunakan.
- Gunakan pautan `Back to top` untuk kembali ke atas dengan cepat.
