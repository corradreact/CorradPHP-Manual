# Upload File / Attachment

Panduan memuat naik dokumen atau lampiran ke sistem.

<div class="main-card">

## Langkah muat naik

1. Buka rekod atau borang yang memerlukan lampiran.
2. Klik **Upload** atau **Add Attachment**.
3. Pilih fail dari komputer anda.
4. Klik **Submit** atau **Save**.
