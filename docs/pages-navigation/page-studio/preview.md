
# Scenario 6: Menggunakan Preview

## Tujuan Scenario

Preview digunakan untuk melihat paparan page sebelum page digunakan oleh user.

## Langkah-langkah

1. Buka page yang ingin disemak dalam **Page Studio**.
2. Pastikan semua perubahan telah disimpan.
3. Klik button **Preview** pada toolbar atas.
4. Semak layout, field, button dan susunan page.
5. Jika ada perubahan diperlukan, kembali ke Page Studio.
6. Edit semula page dan klik **Save**.
7. Preview semula sehingga paparan betul.

- Sentiasa preview selepas membuat perubahan.
- Preview membantu mengesan masalah layout lebih awal.
- Jika data tidak keluar dalam preview, semak backend/API configuration.
