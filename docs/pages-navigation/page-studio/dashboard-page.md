# Scenario 4: Membina Dashboard Page

## Tujuan Scenario

Scenario ini digunakan apabila user ingin membina dashboard untuk memaparkan ringkasan maklumat.

Contoh dashboard:

- Total record
- Status summary
- Chart
- Counter
- Table ringkasan
- KPI card

## Langkah-langkah

1. Buka **Page Studio**.
2. Create page baharu atau pilih dashboard page sedia ada.
3. Tambah component dashboard seperti:
   - Summary card
   - Counter
   - Chart
   - Table
   - Status indicator
4. Susun component supaya paparan kemas.
5. Tetapkan title untuk setiap card atau chart.
6. Sambungkan dashboard kepada data source atau API jika menggunakan data sebenar.
7. Klik **Save**.
8. Klik **Preview** untuk semak layout dashboard.
9. Uji paparan dashboard dengan data sebenar atau sample data.

## Nota Penting

- Paparkan maklumat yang penting sahaja.
- Jangan letak terlalu banyak chart dalam satu page.
- Dashboard yang menggunakan data sebenar memerlukan API atau backend data source.
- Pastikan user mempunyai permission untuk melihat data dashboard.
