
# Scenario 7: Duplicate Page

## Tujuan Scenario

Duplicate Page digunakan apabila user ingin menyalin struktur page sedia ada untuk dijadikan page baharu.

Ini berguna jika page baharu mempunyai layout yang hampir sama dengan page lama.

## Langkah-langkah

1. Buka **Page Studio**.
2. Pilih page yang ingin disalin.
3. Klik button **Duplicate Page**.
4. Masukkan nama page baharu.
5. Kemaskini Menu Title, Page Title dan Breadcrumb.
6. Ubah field atau component jika perlu.
7. Klik **Save**.
8. Klik **Preview** untuk semak page baharu.

## Nota Penting

- Pastikan nama page baharu tidak sama dengan page asal.
- Semak semula field mapping selepas duplicate.
- Jika page asal mempunyai backend logic, pastikan logic untuk page baharu dikemaskini.
