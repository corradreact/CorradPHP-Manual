# Senario 5: Membina Report Page

## Tujuan Senario

Senario ini digunakan apabila user ingin membina page report yang mempunyai filter, table dan fungsi export/download.

Contoh report:

- Report transaksi
- Report user activity
- Report status
- Report audit
- Report summary bulanan

## Langkah-langkah

1. Buka **Page Studio**.
2. Create page baharu untuk report.
3. Tambah filter yang diperlukan seperti:
   - Date From
   - Date To
   - Status
   - Department
   - Category
4. Tambah table atau report component.
5. Susun layout filter dan table dengan kemas.
6. Sambungkan report kepada data source, query atau API.
7. Tambah button export/download jika diperlukan.
8. Klik **Save**.
9. Klik **Preview**.
10. Uji report menggunakan sample data.
11. Semak sama ada filter dan export berfungsi dengan betul.

## Hasil Yang Dijangka

## Nota Penting

- Report biasanya memerlukan backend query atau API.
- Pastikan filter mudah difahami.
- Semak performance jika data terlalu banyak.
- Pastikan permission export diberikan kepada role yang betul sahaja.
