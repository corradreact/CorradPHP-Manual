# System Administrator - Overview

## Purpose

Bagian System Administrator menyediakan alat dan pengaturan untuk mengelola sistem secara keseluruhan. Di sini administrator dapat melihat status sistem, sesi pengguna, dan konfigurasi dasar.

## When to Use

Gunakan modul ini untuk:
- Memantau status sistem
- Melihat siapa yang sedang login
- Mengelola sesi aktif
- Mengakses pengaturan dasar sistem

## Key Features

1. **System Status** - Melihat status keseluruhan sistem
2. **Active Sessions** - Lihat siapa yang sedang login
3. **System Logs** - Melihat aktivitas sistem
4. **Performance Metrics** - Monitor performa
